# 🚧 구라방지턱 (GuraBangjituck)

> **"나 거의 다 왔어!" 이제 그 거짓말(구라)은 통하지 않습니다.**
> 약속 시간에 늦거나 거짓말하는 사람들을 위한 확실하고 무자비한 위치 인증 & 페널티 앱 👊

## 📝 프로젝트 소개

**구라방지턱**은 사용자가 설정한 약속 시간에 정확히 위치를 검사하여, 약속 장소에 도착하지 않았을 경우 지정된 사람에게 **자동으로 페널티 문자(SMS)를 발송**하는 안드로이드 애플리케이션입니다. 앱이 주머니 속에서 꺼져 있더라도, 안드로이드 시스템 알람과 백그라운드 위치 추적을 통해 칼같이 검사를 수행합니다.

## ✨ 주요 기능

* ⏰ **칼 같은 정각 검사:** 안드로이드 `AlarmManager`의 정확한 알람을 사용하여 1분의 오차도 없이 약속 시간을 체크합니다.
* 📍 **무자비한 백그라운드 위치 추적:** 앱을 끄거나 화면을 덮어두어도, 백그라운드에서 사용자의 실제 위치를 파악합니다.
* 🗺️ **네이버 지도 연동:** 직관적인 지도 UI를 통해 약속 장소를 쉽게 지정하고 현재 위치를 확인할 수 있습니다.
* 💸 **자동 페널티 문자 발송:** 조건 미달(지각) 시, 변명할 틈도 없이 등록된 번호로 '구라 적발' 문자를 전송합니다.

## ⚙️ 작동 원리 (Flow)

1. 사용자가 앱을 켜고 **약속 시간, 장소, 페널티 문자 수신자**를 설정합니다.
2. 앱이 포그라운드 서비스(`Foreground Service`)로 전환되며 알람 대기 상태에 들어갑니다.
3. 약속 시간 정각! `GuraCheckReceiver`가 백그라운드에서 깨어납니다.
4. 사용자의 현재 GPS 위치를 가져와 약속 장소 반경과 비교(구라 판정)합니다.
5. 반경 내에 없다면 `SEND_SMS` 권한을 통해 가차 없이 페널티 문자를 쏩니다.

## 🛠 기술 스택

* **OS:** Android (Target API 31 / Android 12)
* **Language:** [Kotlin / Java 중 선택하여 기재]
* **API:** Naver Maps API (`com.naver.maps`)
* **Core Components:** `BroadcastReceiver`, `AlarmManager`, `Foreground Service`

## 🔐 주요 사용 권한 (Permissions)

앱의 강력한 기능을 위해 아래와 같은 핵심 시스템 권한을 사용합니다.

* **위치 (Location):** `ACCESS_FINE_LOCATION`, `ACCESS_BACKGROUND_LOCATION` (실시간 및 백그라운드 GPS 추적)
* **알람 (Alarm):** `SCHEDULE_EXACT_ALARM`, `USE_EXACT_ALARM` (정확한 시간 앱 기상)
* **문자 (SMS):** `SEND_SMS` (자동 페널티 문자 발송)
* **기타:** `FOREGROUND_SERVICE`, `POST_NOTIFICATIONS`, `INTERNET`

2. **일정 등록 화면**
   - **- 화면 상단에서 하단 순서대로 기능 설명드리겠습니다.**
   - 휴대폰번호 입력란: 약속 상대방의 휴대폰번호를 등록합니다. 만약 제시간에 집을 탈출하지 못한 경우에는 여기에 입력된 상대방 휴대폰번호로 지각 사실을 알리는 SMS가 강제로 전송됩니다.
   - 날짜 및 시간 선택란: 언제까지 집을 나서야 약속 시간에 늦지 않는지를 등록합니다. 날짜 및 시간 순서로 선택하게 되어 있습니다. 특징으로는 안드로이드의 순정 dialog를 사용하지 않고, 더 사용자 친화적인 UI/UX를 커스텀하여 적용하였습니다.
   - 굴욕 메시지 입력란: 제시간에 집 탈출 실패 시, 상대방에게 전송되는 SMS에 포함될 메시지 일부를 직접 작성합니다. '랜덤 생성' 버튼을 이용해서 문구를 추천받을 수도 있습니다.
   - 나의 집 위치 설정: 집 탈출 여부를 검증하기 위해 나의 집 위치를 등록해야 합니다. 검증 기준은 설정된 집 위치에서 50m 이상을 벗어났는지 여부로 판단합니다. 특징으로는 해당 상황에서 네이버 지도 API를 활용하게 됩니다.
